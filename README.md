# Installation

```shell
npm install
```

# Compile
```shell
npx hardhat compile
```

# Deploy
```shell
npx hardhat run scripts/deploy.js --network ftm
```
# Deploy Seaport
```shell
npx hardhat run scripts/deploySeaport.js --network goerli
```
